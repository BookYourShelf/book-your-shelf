## Technologies Used

### Back-end
- Spring Boot
- Spring Security
- JPA
- MySQL
- Maven

### Front-end
- Thymeleaf
- Ajax
- jQuery
- Bootstrap


## Requirements

- `Java 11`
- `MySQL 5.7`


## Run

- Create new MySQL database with name `bys_db`
- Download latest release on releases tab
- Run server with `java -jar bookyourshelf-1.0.jar`
